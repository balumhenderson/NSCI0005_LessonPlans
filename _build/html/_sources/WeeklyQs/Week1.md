1. c) Find $i^5$.

```

For this question, it is good to notice the pattern of $i^n$. $i^1 = i$, $i^2 = -1$, $i^3 = -i$, $i^4 = 1$, and then this repeats. 

From here, we can see that $i^{25} = i^1 = i$
```
